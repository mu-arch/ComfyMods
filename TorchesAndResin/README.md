# Torches and Resin

  * Any newly placed `Standing wood torch`, `Standing iron torch` or `Sconce` starts with `10,000` resin fuel.

## Instructions

  * Unzip `TorchesAndResin.dll` to `/Valheim/BepInEx/plugins/`.
